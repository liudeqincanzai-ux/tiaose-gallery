# 怎么修改这个网站

现在网站已经改成 Decap CMS 在线后台。

## 在线修改

打开：

```text
https://你的网址/admin/
```

登录 GitHub 后，可以在线改：

- 网站名称
- 首页封面
- 主题颜色
- 精选作品
- 主题故事
- 全部作品展示方式
- 联系方式
- 上传新照片

点击保存后，后台会把修改提交到 GitHub。网站重新部署完成后，别人打开公开 HTTPS 网址也能看到更新。

通常需要等 1-5 分钟。

## 增加新照片

在 `/admin/` 后台的图片字段里上传即可。新图片会保存到：

```text
photos/
```

## 必须配置的一次性步骤

Decap CMS 要真正保存到 GitHub，需要 Netlify 提供 GitHub OAuth 登录。

需要做一次：

1. 用 Netlify 连接这个 GitHub 仓库并部署
2. 在 GitHub 创建 OAuth App，回调地址填：

```text
https://api.netlify.com/auth/done
```

3. 把 GitHub OAuth 的 Client ID 和 Client Secret 填到 Netlify 的 OAuth 设置里

完成后，打开 `/admin/` 就可以在线修改并保存。

## 文件位置

- `content/site.json`：网站内容
- `admin/config.yml`：后台表单
- `script.js`：前台读取内容并渲染页面
- `style.css`：视觉样式
